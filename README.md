# 🎓 LMS Backend

This is the **backend API** of the LMS (Learning Management System) project, developed using **Laravel**. It powers features such as user authentication, video progress tracking, note-taking, and more — designed to organize and enhance learning from YouTube and other video sources.

---

## 🚀 Tech Stack

- **Framework**: Laravel 10+
- **Language**: PHP 8.1+
- **Database**: PostgreSQL
- **API Format**: RESTful JSON APIs
- **Authentication**: JWT
- **Video Integration**: YouTube Data API (planned)
- **AI Integration**: Gemini API (for summaries and quizzes)
